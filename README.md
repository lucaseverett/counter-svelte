# Counter (Svelte)

This is a basic implementation of a counter using [Svelte](https://svelte.dev/).

## Live Demo

https://code.lucaseverett.dev/counter-svelte/

## Getting Started

Clone this repo:

```sh
git clone https://github.com/lucaseverett/counter-svelte.git
```

Install NPM dependencies:

```sh
npm install
```

Start dev server:

```sh
npm start
```
